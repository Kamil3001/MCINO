What is a Cyclomatic Complexity smell?

Cyclomatic complexity is a software metric used to indicate the complexity of a program. It is a quantitative measure of the number of linearly independent paths through a program's source code. It was developed by Thomas J. McCabe

How to treat?

  • If cyclomatic complexity exceeds 10, then the function needs to be broken up into smaller functions for simplification.

Why bother?

  • Improves understanding and organization of code.
  • Reduces code size.
