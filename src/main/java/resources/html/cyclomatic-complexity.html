What is a Cyclomatic Complexity smell?

Cyclomatic complexity is a way of measuring the complexity of a program. This method was developed by Thomas J. McCabe.

How do we treat cyclomatic complexity?

  • If cyclomatic complexity exceeds a certain point, we then need to divide our functions into smaller ones.

Why do we bother fixing it?

  • Improves the structure of code.
  • Reduces number of lines in code.
