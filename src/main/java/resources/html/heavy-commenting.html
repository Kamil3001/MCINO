What is a Heavy Commenting smell?

When your program is filled with self explanatory comments.

How to treat the heavy commenting smell?

  • Use Extract Variable to split comments that describe complex expressions into subexpressions.
  • Instead of explaining a method, give it a self explanatory name you can use Rename Method for this.
  • To assert rules about a certain state that is needed for the system to work use Introduce Assertion.

Why do we bother fixing this?

  • Code becomes more obvious and easy to understand.
